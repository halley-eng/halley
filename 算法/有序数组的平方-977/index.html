有序数组的平方 977

October 16, 2020

给定一个按非递减顺序排序的整数数组 A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

 

示例 1：

输入：[-4,-1,0,3,10] 输出：[0,1,9,16,100] 示例 2：

输入：[-7,-3,2,3,11] 输出：[4,9,9,49,121]  

提示：

  1. 1 <= A.length <= 10000
  2. -10000 <= A[i] <= 10000
  3. A 已按非递减顺序排序。

题解

直接排序


class Solution {
    public int[] sortedSquares(int[] A) {
        int len = A.length;
        for(int i = 0; i < len;i++){
            A[i] = A[i] * A[i];
        }
        Arrays.sort(A);
        return A;
    }
}

时间复杂度: O(nlogn) 空间复杂度: O(1)

双指针

非递减顺序排序的整数数组 A 的平方特征是 两边较大, 中间较小;


class Solution {
    public int[] sortedSquares(int[] A) {
        
        int len = A.length;
        int[] ans = new int[len];
        int pos = len - 1, left = 0, right = len - 1;
        while(left <= right){
            int l = A[left] * A[left], r = A[right] * A[right];
            if(l > r){
                ans[pos--] = l;
                left++;
            }else{
                ans[pos--] = r;
                right--;
            }
        }

        return ans;
    }
}

时间复杂度 O(n) 空间复杂度 O(n)
